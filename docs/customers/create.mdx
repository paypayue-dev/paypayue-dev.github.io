# Criar cliente

Esta operação permite registar o cliente na PayPay. Recomenda-se a utilização do parâmetro `customerId` para evitar registos duplicados.

Criação de um cliente com os seguintes dados:

- **Identificador do cliente:** 1
- **Primeiro nome:** José
- **Último nome:** Silva
- **Email:** jose.silva@email.com
- **Número de telefone:** 351#961234567

> **POST** /customers
