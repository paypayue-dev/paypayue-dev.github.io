# Associar o cliente a outras entidades

> **POST** /customers/_\{uuid}_/entities

A associação de um cliente a outras entidades, permite que o registo do cliente fique disponível nas entidades desejadas.

Para realizar essa associação é necessário conhecer o _uuid_ do cliente e o _clientId_ das entidades.

**1.** Associar o cliente com o _uuid_ e2343605-cf46-43de-b20b-9b7d1c95a9b2 a entidade com o _clientId_ 523308191 e 559514166
