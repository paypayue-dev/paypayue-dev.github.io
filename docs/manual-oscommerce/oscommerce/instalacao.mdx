# Instalação

Realize o download do módulo osCommerce da PayPay disponível em https://paypay.pt/paypay/#integracoes. De seguida deve extrair os ficheiros do zip e copiar os ficheiros fornecidos para a pasta "catalog" que contém os ficheiros de instalação da sua loja osCommerce, como ilustra a figura seguinte.

![instalacao1](../../../_media/oscommerce-2.png)

Após todos os ficheiros terem sido copiados com sucesso, aceda à sua área de administração no menu dos módulos de pagamento. Clique em "Install module" para visualizar os módulos a instalar, entre eles deve constar os seguintes módulos da PayPay:
- PayPay - Core Module (obrigatório);
- PayPay - Credit/DebitCard;
- PayPay - Multibanco;
- PayPay - MB WAY

![instalacao2](../../../_media/oscommerce-3.png)

Selecione o módulo PayPay - Core Module e clique em "Install Module". Este módulo é obrigatório, para os restantes pode selecionar e instalar os que pretende utilizar.

![instalacao3](../../../_media/oscommerce-4.png)
