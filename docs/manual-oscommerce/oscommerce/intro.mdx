# Requisitos

<a href="https://apps.oscommerce.com/Apps&JHMvS&paypay-oscommerce" target="_blank" className="btn-api btn-api--float-right">Descarregar</a>

- PHP 5.4
- osCommerce 2.3.2
- [SOAP extension](http://php.net/manual/en/book.soap.php)

Após a instalação concluída e a loja configurada, aceda à área de administração e efetue o
seu login.
