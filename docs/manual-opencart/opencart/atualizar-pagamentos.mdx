# Atualizar os pagamentos manualmente

Todos os pagamentos são processados automaticamente. No entanto, caso verifique alguma situação em que o pagamento foi efetuado, mas não foi atualizado na loja OpenCart, o administrador da loja poderá utilizar o botão ‘Verify Payments’ para atualizar manualmente todos os pagamentos pendentes. Este botão está disponível no separador de módulos de pagamentos ‘Payments’ do ‘Dashboard’.<br/> Altere também o ‘Status’ de pagamento.
