# Restrições por país/transportadora/etc

Ao aceder as preferências de pagamento do Prestashop é possível configurar que métodos depagamento devem estar disponíveis para os seus clientes dependendo da moeda, país, perfil de utilizador ou transportadora.

![Restrições](../../../_media/prestashop-19.png)

## Restrições de Moeda

Selecionar os módulos disponíveis por tipo de moeda que se aplique:

Os módulos PayPay apenas suportam EUR

:::
