# Requisitos

<a href="https://paypay.pt/paypay/public/plugins/prestashop-plugins.zip" target="_blank" className="btn-api btn-api--float-right">Descarregar</a>

Para executar a integração com a PayPay:
- O comerciante deverá estar registado na plataforma PayPay;
- Deverá dispor de uma loja online PrestaShop:
    https://www.prestashop.com/pt/;
- Deverá descarregar as pastas zipadas em https://paypay.pt do Prestashopeguardar no seu computador;
- Entrar no ambiente de administração da sua plataforma de e-commerce.
