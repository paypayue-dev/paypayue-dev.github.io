# Listado de clientes

> **GET** /customers

El listado de clientes se puede realizar utilizando los siguientes filtros:

- **Correo electrónico** (email)
- **Identificador del cliente** (customerId)
- **Limitar los resultados** (limit)

**1.** Obtener el cliente con el correo electrónico jose.silva@email.com.
