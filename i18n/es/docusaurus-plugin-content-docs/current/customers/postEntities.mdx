# Asociar el cliente a otras entidades

> **POST** /customers/_\{uuid}_/entities

La asociación de un cliente con otras entidades, permite que el registro del cliente esté disponible en las entidades deseadas.

Para realizar esta asociación es necesario conocer el _uuid_ del cliente y el _clientId_ de las entidades.

**1.** Asociar el cliente con el _uuid_ e2343605-cf46-43de-b20b-9b7d1c95a9b2 a la entidad con el _clientId_ 523308191 e 559514166
