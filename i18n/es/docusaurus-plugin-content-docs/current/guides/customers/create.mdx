# Crear cliente

Esta operación permite registrar al cliente en PayPay. Se recomienda utilizar el parámetro `customerId` para evitar registros duplicados.

Crear un cliente con los siguientes datos:

- **Identificación del cliente:** 1
- **Nombre:** José
- **Apellido:** Silva
- **Correo electrónico:** jose.silva@email.com
- **Número de teléfono:** 351#961234567

> **POST** /customers
