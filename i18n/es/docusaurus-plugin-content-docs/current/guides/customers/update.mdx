# Actualización del cliente

> **PUT** /customers/_\{uuid}_

La siguiente información puede ser actualizada:

- **Identificador del cliente** (customerId)
- **Nombre** (firstName)
- **Apellido** (lastName)
- **Correo electrónico** (email)
- **Número de teléfono** (phoneNumber)

**1.** Actualizar el número de teléfono del cliente.
